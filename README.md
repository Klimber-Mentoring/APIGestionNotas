API REST que permite administrar notas.

Una nota contiene los siguientes campos:
- Id
- Título
- Contenido
- Fecha de creación
- Fecha de modificación

Endpoints:
- Obtener todas las notas [GetAll]

- Obtener nota por id [GetById(Guid id)]

- Crear una nota [Create]

- Actualizar una nota [Update]

- Eliminar una nota [Delete]
